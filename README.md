# Trip Vault

A personal, offline-capable trip command center — store flights, hotels, documents,
tickets, and budgets in one secure place and pull them up on your phone.

Built with React + Vite + Tailwind, Supabase (auth, Postgres, storage), and a
Workbox-powered PWA for offline use. Deployed on Vercel.

## Getting started

```bash
npm install
cp .env.example .env.local   # fill in your Supabase URL + anon key
npm run dev
```

See `nanny_trip_vault_PRD.md` for the full product spec and build phases.
